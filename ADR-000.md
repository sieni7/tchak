# ADR-000 – Décisions Architecturales Fondatrices

## Décisions prises (après prototype validé)

1. **Stack de base** : HTML/CSS/JS pur (pas de framework lourd)
2. **Hachage** : Web Crypto API (SHA256) – natif navigateur
3. **Signature** : Canvas avec export PNG
4. **Export PDF** : `window.print()` (pas de librairie externe)
5. **Persistance** : localStorage (pas de base de données pour V0)
6. **Distribution** : Fichier HTML unique (ZIP)

## Décisions reportées

- Electron (post-V0)
- Cloud / API (version PRO)
- Base de données structurée (SQLite / Supabase)
