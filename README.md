# TCHAK ! — Certification d'Authenticité d'Art

TCHAK ! est un écosystème de certification pour les artistes peintres, permettant d'émettre des certificats d'authenticité sécurisés par une chaîne de confiance SHA256 déterministe.

## 🚀 Version FREE V0 (Disponible)
- **Onboarding Cloud** : Inscrivez-vous sur [tchak.art](https://tchak.art).
- **App Desktop Offline** : Générez vos certificats sans connexion internet.
- **Quota** : 10 certificats gratuits par mois.
- **Sécurité** : Intégrité locale garantie.

## 📂 Structure du Projet
- `landing/` : Site web d'onboarding et de distribution.
- `electron-app/` : Application desktop multi-plateforme.
- `supabase/` : Schéma de données cloud.
- `docs/` : Guides utilisateur et développeur.

## 🛠 Installation Rapide (Dev)
```bash
# Initialisation
./init_project.sh

# Lancement
npm run dev:landing
npm run dev:electron
```
