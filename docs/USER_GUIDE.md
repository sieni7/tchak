# Guide Utilisateur : TCHAK FREE

## 1. Installation
1. Téléchargez le ZIP depuis [tchak.art](https://tchak.art).
2. Extrayez le contenu et lancez `TCHAK-FREE-Setup.exe` (Windows) ou ouvrez le `.dmg` (macOS).
3. Suivez les instructions de l'installeur.

## 2. Création d'un Certificat
1. Lancez l'application **TCHAK FREE**.
2. Cliquez sur **Nouveau Certificat**.
3. Remplissez les informations de l'œuvre (Titre, Artiste, Année, Médium, Dimensions).
4. Choisissez un thème visuel (Classique ou Moderne).
5. Vérifiez la prévisualisation A4.
6. Cliquez sur **Exporter en PDF**. Le fichier sera sauvegardé dans votre dossier **Documents**.

## 3. Partage et Vérification
- Vous pouvez cliquer sur **Partager** pour ouvrir le dossier contenant le PDF.
- L'ID unique (SHA256) en bas du certificat garantit son authenticité technique dans votre chaîne locale.

## 4. Sauvegarde des Données
Il est fortement recommandé de cliquer sur **Exporter les données (ZIP)** dans le dashboard une fois par mois pour conserver une copie de votre chaîne de confiance en dehors de votre ordinateur.
